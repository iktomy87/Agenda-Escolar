# Agenda-Escolar

Aplicación de escritorio destinada a la gestión de tareas académicas, organizadas por materias. Permite al usuario agregar, eliminar y visualizar tareas asignadas a materias específicas. Además, ofrece diferentes vistas donde puede ver todas las tareas en general, agrupadas según su asignatura o agrupadas según la semana o mes. Las tareas se guardan de manera persistente en archivos JSON, permitiendo al usuario guardar y cargar archivos en rutas y con nombres personalizados.
